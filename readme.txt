The plugin prayer during Ramadan.

== Description ==

prayer during Ramadan until 1399 (Solar year).
Persian text of the Prayer Book is Mafatih Aljnan / Publications dehghan.

== Installation ==

1. Upload `month of ramadan` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. drag widget anywhere you wish
